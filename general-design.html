Featured project: Phoenix Ikki - Overview



🏅Phoenix Ikki - Contest finalist 2020 


Process:

  • Model: Blender (based on human body references) 
  • Body: Skin modifier, Blender sculpt tools. 
  • Armor: Solidify, modifiers with mirror, lattice modifiers. 
  • Phoenix tails: Array modifiers with custom ends. 
  • Textured: Substance Painter 


You can view more of the making of this project in my the dedicated Ikki Phoenix page by pressing in the Fire icon or in my ArtStation site.

Featured project: 

Swiss Knife: 


Interactive experience, the Swiss Knife opens and closes, shows Tool Tips, the User Interface is responsive and is connected to the 3D model so it changes animations and materials.

Interactive project that integrates 3D model from Blender, adds a logic layer that interconnects the user interface with the model animations in Blender.


Process:

  • Model: Blender (model provided) , Animated and textured.
  • Responsive UI: Breakpoints with Media Queries in HTML5/CSS3.  Hosted and deployed.
  • Web Optimized: Textures and 3D model optimization for performance. 
  • Programmed Interaction: User Interface interacts with the model and triggers the animation. 
  • Tool Tips: Augmented Reality  annotations.
